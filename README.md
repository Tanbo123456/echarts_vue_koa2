# echarts_vue_koa2
echarts+vue+koa2的数据展示项目

## 前台项目vision， 后台项目echart_server
其分别进行 npm i -y

## 后台项目基于koa2写的简单服务器
### 启动项目
  node app.js
  
## 前台项目基于vue开发的简单应用
### 启动项目 
  yarn serve
  
### 前端项目描述
主要练习echarts实现大屏显示
  1）包括销量趋势图
  2）商家地域分布图
  3）商品销售金额占比统计图
  4）商家销量图
  5）地区销量排名图
  6）库存-销量图
  7）以及大屏联动效果（一端操纵，多端联动）
